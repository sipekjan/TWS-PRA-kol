CSS

CSS je v informatice jazyk pro popis způsobu zobrazení elementů na stránkách napsaných v jazycích HTML.

Aby jsem zapisujeme ve style pak napíšeme co chceme upravit vzhled například h1{ dále napíšeme jak to chceme upravit třeba color: barva red;}

Úpravy

  • color: barva písma;
  • font-family: font písma;
  • font-size: velikost písma(třeba v pixelech 30px);
  • background-color: barva pozadí;
